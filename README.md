## Online chat with React and Firebase
you can try it: https://chat-app-7927b.web.app

1) $ npm i
2) $ cd functions/
3) $ npm i

---- For firebase deploying ----  
- create realtime test database in FireBase  
$ firebase login  
$ firebase init  
$ npm run build - for build  
$ npm start - for start  
$ firebase deploy - for deploy
